Tuning Files

The below tuning files can be used with modular synthesizers with quantiser modules such as the Expert Sleepers Disting range; with VCVRack with modules such as the VCV Scalar module; or with conventional DAWs via compatible VSTs such as Plogue’s Chipsounds/Sforzando.

The .scl file contains the scale/tuning information while the .kbm file tells the instrument what the root note is and how the scale should map to a conventional keyboard. Some synths, like VCV’s Scalar, work with just the .scl file, in which case you will need to manually set your oscillator to the root frequency (which you will find in the .kbm file). Both .scl and .kbm files can be opened and edited with conventional text editors. The below files were created with free online tool Sevish’s Scale Workshop.

The tunings with "Cello" in the filename have been transposed so that the root note is C.
